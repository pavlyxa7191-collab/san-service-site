Скриншоты отзывов для карусели на главной
=====================================
Файлы review-01.png … review-06.png — реальные скриншоты с площадки объявлений.

Чтобы добавить ещё скриншоты:
1. Положите PNG/JPG в эту папку: review-07.png и т.д.
2. В client/src/data/reviewScreenshots.ts добавьте строку:
   { src: "/reviews/review-07.png", alt: "Отзывы клиентов — скриншот 7" },
