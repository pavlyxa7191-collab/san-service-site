# Экоцентр — сайт санитарной службы

## Сборка и запуск

```bash
pnpm install
pnpm run build
pnpm run dev
```

Деплой фронта: артефакты Vite в `dist/public` (см. `vite.config.ts`).

## Меню навигации

Порядок пунктов задаётся **только** в `client/src/components/SiteHeader.tsx` → массив **`navEntries`**:

1. Услуги (выпадающий список — `SERVICE_LINKS`)
2. Цены
3. Отзывы (якорь `#reviews`)
4. Сертификаты (якорь `#certificates`)
5. О компании
6. Блог
7. Контакты

Для десктопа и мобильного меню используется один и тот же массив; у пунктов задан **`order`**, чтобы порядок не ломался вёрсткой.

**Важно:** собирайте проект из **этого каталога** (`san-service-site`), а не из устаревших копий в `Downloads` — иначе на сайте может оказаться старая версия шапки.
